Building Blocks

Module 3 - Creative Technology

Overview

We were asked by BlocksLXP, a company that creates customized e-learning platforms, to create a prototype that can aid in explaining what their company does to potential customers.

The Prototype

Me and my team created physical puzzle pieces. These pieces each contained an esp32, multiple hall effect sensors and magnets. We also coded a simulated website using processing. Each puzzle piece represented a module used by blocksLXP, and users could use our prototype to learn more about how each module functions.
